# Sokrat — one-pager для бізнесу

**Що це.** Self-hostable AI-агент-репетитор, який навчає студентів із матеріалів
школи, памʼятає кожного учня, веде сократівськи (не «зливає» відповідь) і
передає складні випадки живому куратору. Не чат-обгортка над GPT, а агент із
інструментами та памʼяттю.

## Проблема
- Куратори/викладачі тонуть у **повторюваних питаннях** студентів 24/7.
- «Гола» GPT-обгортка **галюцинує** і вчить студентів вигадками — недопустимо.
- Звичайні боти **не памʼятають** учня і не адаптуються.

## Рішення → ефект
| | Без Sokrat | З Sokrat |
|---|---|---|
| Рутинні питання | вручну, в робочий час | автоматично, 24/7, із матеріалів школи |
| Достовірність | ризик галюцинацій | відповіді заземлені + цитати джерел |
| Персоналізація | немає | памʼять учня, адаптація складності |
| Видимість для викладача | читати всі діалоги | 1 авто-звіт на студента + ескалації |

## ROI (ілюстративно — підставте свої числа)
Припущення: 5 000 активних студентів, ~4 звернення/міс, модель `gpt-4o-mini`.

| Метрика | Оцінка |
|---|---|
| Звернень на місяць | ~20 000 |
| Вартість AI (~$0.002/звернення) | **~$40 / міс** |
| Якби кожне питання займало ~4 хв куратора | ~1 300 год/міс ручної роботи |
| При консервативному дефлекті 40–60% | **~530–800 год/міс звільнено** |

> Навіть за найобережніших припущень вартість AI — десятки доларів, а економія —
> сотні людино-годин, які куратори витрачають на складні кейси, а не на «де кнопка».

## Чому це масштабується (інженерна частина)
- **Structured Outputs + Pydantic** — стабільні, передбачувані відповіді.
- **RAG з цитатами** — навчання лише з матеріалів школи, без вигадок.
- **Памʼять учня** як окремий шар — легко перенести на Postgres.
- **Векторний пошук** абстрагований — заміна на pgvector/Qdrant без переписування.
- **Кеш, ретраї, облік токенів і вартості** — готовність до продакшену.
- **Human-in-the-loop** — ескалації в лог для куратора.

## Дорожня карта

дашборд для кураторів.
